# COMP5530M

# Prerequisites

1. CMake
2. sqlite3
3. Qt

## How to run

1. cd to this repo
2. Type in 'mkdir build && cd build'
3. Type in 'cmake ../.'
   - This will Generate build files inside this build folder
4. Type in 'cmake --build .'. Solution has been built.
